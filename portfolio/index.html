Raw Thought: The Weblog of Aaron Swartz

  • Ruby
  • Pandoc
  • Markdown
  • LaTeX
  • ePub

GitHub

I wrote a set of Ruby scripts to scrape the late Aaron Swartz's blog and convert the contents into Markdown format. From the Markdown I used Pandoc to generate PDF, ePub, and Mobi files for each post; I also wrote a script to join all the posts together into one book-like file that's also available in all those formats.

Kronos.js

  • CoffeeScript
  • Node.js
  • Mocha
  • AMD
  • Klingons

GitHub

Kronos is a side project of mine. It's a date/time library that aims to provide a natural syntax with extensible natural language support. It's written in CoffeeScript and was TDD'd using Mocha and Node.js. It's incredibly incomplete right now but has some convenient tidbits, such as a better timeout/interval syntax:

Kronos.in("5 seconds").do(-> alert "Could not reach server").unless(-> server.reached)
